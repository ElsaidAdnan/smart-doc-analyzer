# 🤖 مساعد الوثائق الذكي (Smart Document Analyzer)

تطبيق Streamlit مدعوم بـ **RAG + LangGraph** لتحليل ومقارنة الوثائق (PDF) بذكاء.

## المميزات
- رفع وتحليل متعدد الملفات PDF
- استخراج الجداول والنصوص بجودة عالية باستخدام LlamaParse
- نظام وكلاء متعدد (Extractor → Analyst → Critic)
- دعم كامل للغة العربية في الـ PDF والـ TXT
- تصدير التقارير (PDF + TXT) لآخر سؤال أو المحادثة كاملة
- رسوم بيانية تفاعلية
